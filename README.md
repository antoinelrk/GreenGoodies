# Green Goodies

## Description du projet

Green Goodies est une application web dédiée à la promotion de produits écologiques et durables. Elle vise à sensibiliser les utilisateurs aux enjeux environnementaux tout en leur offrant une plateforme pour découvrir et acheter des produits respectueux de l'environnement.

## Setup

**Pré-requis**

- Avoir Docker et Docker Compose installés sur votre machine.

Après avoir cloné le dépôt, lancez la commande ``make setup`` pour initialiser et démarrer la stack. Aucune action n'est requise.

> Attention : Ce projet n'est pas destiné à être déployé en production. Il est uniquement conçu à des fins éducatives et de démonstration.
> Le fichier .env à été ajouté afin de faciliter le setup. Dans le cadre de la mise en production, il est recommandé d'utiliser des variables d'environnement sécurisées.

Un utilisateur par défaut est créé lors du setup avec les informations suivantes :
- Email : ``john@doe.fr``
- Mot de passe : ``P@ss1234``
